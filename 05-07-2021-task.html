Difference between Scripting Languages and Programming Languages ?

The main difference is programming Languages uses compliler to convert the code in to machine executable code and also it checks the entire code at a time if we get one error complete code is not executed.
Scripting Languages uses interpreter to convert the code in to machine executable code and also it checks the code line by line even if we get an error code is executed till the line where we got the error.
